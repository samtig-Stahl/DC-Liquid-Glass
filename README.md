# DC Liquid Glass - Vencord CSS

**DC Liquid Glass** ist ein modisches und benutzerdefiniertes CSS-Design für **Vencord**, das deinem Discord-Client ein stilvolles und modernes Aussehen verleiht. Dieses CSS nutzt die Liquid-Glass-Technologie, um eine transparente, fließende Oberfläche zu schaffen, die sowohl ästhetisch als auch funktional ist.

## Features

- **Flüssigglas-Effekt**: Schafft eine beeindruckende, durchscheinende Oberfläche, die an Liquid-Glass erinnert.
- **Vencord-Unterstützung**: Vollständig kompatibel mit dem Vencord-Client, einem beliebten Discord-Customizer.
- **Responsives Design**: Optimiert für verschiedene Bildschirmgrößen und -auflösungen.
- **Leichtgewichtig und Performant**: Das Design beeinträchtigt kaum die Leistung deines Discord-Clients.

## Installation

### Voraussetzungen

Bevor du mit der Installation beginnst, stelle sicher, dass du **Vencord** auf deinem Computer installiert hast. Falls du Vencord noch nicht installiert hast, kannst du die [Vencord-Installationsanleitung](https://github.com/vencord/vencord) einsehen.

### Schritte zur Installation

1. **CSS-Datei herunterladen**

   Lade die neueste Version von **DC Liquid Glass** herunter:

   - Gehe auf die [Releases-Seite](https://github.com/samtig-Stahl/DC-Liquid-Glass/releases) und lade die `system32.css`-Datei herunter.

2. **CSS in Vencord hinzufügen**

   - Öffne Vencord und gehe zu den Einstellungen.
   - Navigiere zu **"Themes"** und lade die heruntergeladene `system32.css`-Datei in den Themes Ordner.
   - Speichere deine Änderungen und starte Discord neu, um den flüssigglasartigen Effekt zu sehen.
